# 检索表格弹窗标准模板

> 适用于常见后台列表页：上方检索，中间操作按钮，下方表格，新增/编辑使用弹窗表单。

## 使用场景

- 列表页需要查询、重置、分页、刷新。
- 表格需要操作列，例如编辑、删除。
- 新增/编辑用同一个弹窗表单承载。

## 组件组合

- 检索区：`Search + FormSchema`
- 表格区：`Table + useTable`
- 弹窗区：`Dialog + Form + useForm`
- 页面容器：`ContentWrap`

## 标准骨架

```vue
<script setup lang="tsx">
import { computed, nextTick, reactive, ref, unref } from 'vue'
import { ElButton, ElTag } from 'element-plus'
import { ContentWrap } from '@/components/ContentWrap'
import { Dialog } from '@/components/Dialog'
import { Form, type FormSchema } from '@/components/Form'
import { Search } from '@/components/Search'
import { Table, type TableColumn } from '@/components/Table'
import { useForm } from '@/hooks/web/useForm'
import { useTable } from '@/hooks/web/useTable'

type Status = 'enabled' | 'disabled'

interface ListItem {
  id: number
  name: string
  status: Status
  createTime: string
}

interface QueryParams {
  keyword?: string
  status?: Status
}

const statusOptions = [
  { label: '启用', value: 'enabled' },
  { label: '禁用', value: 'disabled' }
]

const queryParams = ref<QueryParams>({})
const dialogVisible = ref(false)
const dialogType = ref<'add' | 'edit'>('add')
const editRecord = ref<ListItem>()
const dialogTitle = computed(() => (dialogType.value === 'add' ? '新增' : '编辑'))

const { formRegister, formMethods } = useForm()
const dialogFormColProps = { span: 24 }

const searchSchema = reactive<FormSchema[]>([
  {
    field: 'keyword',
    label: '关键词',
    component: 'Input',
    componentProps: { placeholder: '请输入关键词' }
  },
  {
    field: 'status',
    label: '状态',
    component: 'Select',
    componentProps: {
      placeholder: '全部状态',
      options: statusOptions
    }
  }
])

const formSchema = reactive<FormSchema[]>([
  {
    field: 'name',
    label: '名称',
    component: 'Input',
    colProps: dialogFormColProps,
    formItemProps: {
      rules: [{ required: true, message: '请输入名称', trigger: 'blur' }]
    }
  },
  {
    field: 'status',
    label: '状态',
    component: 'Select',
    colProps: dialogFormColProps,
    formItemProps: {
      rules: [{ required: true, message: '请选择状态', trigger: 'change' }]
    },
    componentProps: { options: statusOptions }
  }
])

const columns = reactive<TableColumn[]>([
  { field: 'index', label: '序号', type: 'index', width: 70 },
  { field: 'name', label: '名称', minWidth: 160 },
  {
    field: 'status',
    label: '状态',
    width: 100,
    slots: {
      default: ({ row }) => (
        <ElTag type={row.status === 'enabled' ? 'success' : 'danger'}>
          {row.status === 'enabled' ? '启用' : '禁用'}
        </ElTag>
      )
    }
  },
  { field: 'createTime', label: '创建时间', width: 140 },
  {
    field: 'action',
    label: '操作',
    width: 120,
    fixed: 'right',
    slots: {
      default: ({ row }) => (
        <ElButton type="primary" link onClick={() => openEditDialog(row as ListItem)}>
          编辑
        </ElButton>
      )
    }
  }
])

const { tableRegister, tableState, tableMethods } = useTable({
  immediate: true,
  fetchDataApi: async () => {
    // 替换为真实接口：getListApi({ ...queryParams.value, page, pageSize })
    return { list: [], total: 0 }
  }
})

const reloadTable = () => tableMethods.getList()

const searchList = (params: QueryParams) => {
  queryParams.value = params
  tableState.currentPage.value = 1
  reloadTable()
}

const resetList = (params: QueryParams) => {
  queryParams.value = params
  tableState.currentPage.value = 1
  reloadTable()
}

const openAddDialog = async () => {
  dialogType.value = 'add'
  editRecord.value = undefined
  dialogVisible.value = true
  await nextTick()
  const elForm = await formMethods.getElFormExpose()
  elForm?.resetFields()
}

const openEditDialog = async (row: ListItem) => {
  dialogType.value = 'edit'
  editRecord.value = row
  dialogVisible.value = true
  await nextTick()
  await formMethods.setValues(row)
  const elForm = await formMethods.getElFormExpose()
  elForm?.clearValidate()
}

const submit = async () => {
  const elForm = await formMethods.getElFormExpose()
  await elForm?.validate(async (isValid) => {
    if (!isValid) return
    const formData = await formMethods.getFormData()
    // dialogType.value === 'add' ? createApi(formData) : updateApi({ id: editRecord.value?.id, ...formData })
    dialogVisible.value = false
    reloadTable()
  })
}
</script>

<template>
  <ContentWrap class="mb-12px">
    <Search
      :schema="searchSchema"
      :is-col="true"
      :column-count="4"
      label-width="80px"
      @search="searchList"
      @reset="resetList"
    />
  </ContentWrap>

  <ContentWrap>
    <div class="mb-12px flex items-center justify-between">
      <BaseButton type="primary" @click="openAddDialog">新增</BaseButton>
    </div>

    <Table
      v-model:current-page="tableState.currentPage.value"
      v-model:page-size="tableState.pageSize.value"
      row-key="id"
      :columns="columns"
      :data="tableState.dataList.value"
      :loading="tableState.loading.value"
      :pagination="{ total: tableState.total.value }"
      show-action
      stripe
      @register="tableRegister"
      @refresh="reloadTable"
    />
  </ContentWrap>

  <Dialog v-model="dialogVisible" :title="dialogTitle" width="560px" max-height="360px">
    <Form :schema="formSchema" label-width="90px" :is-col="true" @register="formRegister" />

    <template #footer>
      <BaseButton @click="dialogVisible = false">取消</BaseButton>
      <BaseButton type="primary" @click="submit">保存</BaseButton>
    </template>
  </Dialog>
</template>
```

## 开发要点

1. 检索条件变化后，先把 `currentPage` 重置为 `1`，再刷新表格。
2. 搜索项列数优先用 `Search` 的 `columnCount` 或 `responsiveColProps` 控制。
3. 弹窗表单一列展示时，`Form` 设置 `:is-col="true"`，字段设置 `colProps: { span: 24 }`。
4. 编辑回显需要先打开弹窗，再 `nextTick` 后调用 `formMethods.setValues(row)`。
5. 新增打开前清空 `editRecord`，并调用 `resetFields()` 清理旧表单值和校验。
6. 真实接口应拆到 `src/api/业务模块/`，请求参数和返回类型必须显式声明。
